# Railway 환경변수 설정

## 필수 환경변수 (Railway 대시보드의 Variables 탭에서 설정)

NODE_ENV=production
JWT_SECRET=isggo_production_jwt_secret_key_2024_please_change_this_strong_key
PORT=3000

## 선택 환경변수 (필요시 추가)

# TZ=Asia/Seoul (시간대 설정)
# LOG_LEVEL=info (로깅 레벨)

## 설정 방법
1. Railway 프로젝트 → 서비스 클릭
2. Variables 탭 클릭
3. New Variable 버튼으로 위 환경변수들 추가
4. Deploy 탭에서 재배포 확인

## 주의사항
- JWT_SECRET은 반드시 강력한 비밀키로 변경하세요
- PORT는 Railway에서 자동으로 설정되므로 생략 가능
- NODE_ENV=production은 필수입니다
