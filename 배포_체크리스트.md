## 🔧 문제 발생 시 확인 사항

### Railway 로그 확인
1. Railway 대시보드 → 서비스 클릭
2. "Deployments" 탭에서 최신 배포 클릭
3. 로그에서 에러 메시지 확인

### 일반적인 문제들
- **500 에러**: 환경변수 누락 확인
- **파일 못 찾음**: 정적 파일 경로 확인
- **DB 연결 실패**: 데이터 디렉토리 권한 확인
- **JWT 에러**: JWT_SECRET 환경변수 확인
